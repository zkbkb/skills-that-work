# 通用占卜解读方法论

> 本文档是 qi-gua skill 的核心解读参考。
> 不绑定特定占问类型，提供一套通用推理规则（规则层），
> 并在关键节点嵌入场景特异性映射（校准层），
> 由 Claude 根据每次具体占问动态适配。
>
> 规则层解决「分析什么」，校准层解决「分析到什么精度」。
>
> 方法论来源：梅花易数传统理论 + 六爻纳甲京房体系，
> 结合 divination-lost-items skill 的实战验证经验泛化而来。

## 1. 用神推导规则

### 1.1 核心原理

六爻占卜中，**用神**是代表所问之事的那一爻。用神的选择决定了整个分析的基准点。

用神的确定基于**六亲**体系：以卦宫五行为"我"，各爻地支五行与"我"的关系
决定了该爻的六亲属性。不同的占问内容映射到不同的六亲作为用神。

**推导时必须写出推理链**，不可直接跳到结论。

> ⚠️ 注意：当前引擎的 yongshen 输出默认取妻财爻（引擎遗留）。
> 在通用占问中，**必须根据占问内容重新确定用神**，
> 不可直接使用引擎输出的 yongshen 字段——
> 该字段仅在占问恰好以妻财为用神时直接适用。

### 1.2 用神映射规则

#### 官鬼为用神

官鬼代表「克我者」——权威、约束、压力、功名。

| 占问 | 推导理由 |
|------|---------|
| 求职、升迁、仕途 | 官位为克制约束之象 |
| 诉讼、官司 | 司法权威为克我之力 |
| 疾病（问病情走向） | 病为克身之象 |
| 考试（官方认定） | 官方权威的认可 |
| 丈夫（女性问婚姻） | 传统六亲中女以官为夫 |
| 盗贼、威胁来源 | 克我损我者 |
| 上司、领导、权威人物 | 管辖约束我者 |

#### 妻财为用神

妻财代表「我克者」——我能掌控和获取的对象。

| 占问 | 推导理由 |
|------|---------|
| 财运、收入、投资 | 我克者为我之财 |
| 寻物（物品、财物） | 财物为我所有之物 |
| 妻子（男性问婚姻） | 传统六亲中男以财为妻 |
| 买卖、交易 | 交易标的为可获取之物 |
| 天气（问晴） | 财克官（官=雨），财旺则晴 |

#### 父母为用神

父母代表「生我者」——养育、庇护、文书、学问。

| 占问 | 推导理由 |
|------|---------|
| 学业（以知识论） | 文书、知识为生我者 |
| 文件、合同、证书 | 一切文书之象 |
| 房屋、居所 | 庇护容身之所 |
| 父母长辈 | 直接对应 |
| 师长、老师 | 授业解惑生我知识者 |
| 车船、交通工具 | 承载我者 |

> **考试用神的特殊区分：**
> 「能否通过考核」→ 官鬼（官方认可）；
> 「学习效果、知识掌握」→ 父母（文书学问）。
> 模糊时主动询问用户侧重点。

#### 子孙为用神

子孙代表「我生者」——我产出的、养育的、享受的。

| 占问 | 推导理由 |
|------|---------|
| 子女 | 直接对应 |
| 享乐、娱乐、休闲 | 我之乐 |
| 学生、下属 | 我所培育者 |
| 医药、治疗 | 子孙克官鬼（病），解病之象 |
| 宠物 | 我所养育者 |
| 解忧、消灾 | 子孙制官鬼（忧患），解忧之象 |

#### 兄弟为用神

兄弟代表「比和者」——同辈、竞争、合作。

| 占问 | 推导理由 |
|------|---------|
| 兄弟姐妹、朋友、同事 | 同辈同类者 |
| 竞争对手 | 与我争夺资源者 |
| 合伙人 | 同辈协作者 |

> **注意：** 兄弟在多数占问中是忌神（克妻财=耗财）。
> 只有问兄弟朋友本人的事时才以兄弟为用神。

### 1.3 复合占问的用神处理

占问涉及多个维度时（如"跳槽能赚到钱吗"同时涉及求官和求财）：

1. **识别主要关切** → 确定主用神
2. **主用神 + 参考用神**：主要关切定主用神，次要关切定参考用神
3. **无法判断时主动询问**

### 1.4 用神不上卦（伏藏）

用神对应的六亲在卦中没有出现时，用神处于**伏藏**状态。
详见第 8 节「伏神分析」。

## 2. 六亲通用释义

### 2.1 释义矩阵

| 六亲 | 核心含义 | 人物象 | 事物象 | 卦中角色 |
|------|---------|--------|--------|---------|
| **兄弟** | 比和、同类 | 兄弟姐妹、同事、朋友、竞争者 | 阻碍、竞争、消耗 | 克妻财（劫财）、生子孙 |
| **子孙** | 我生、产出 | 子女、学生、下属 | 享乐、解决方案 | 克官鬼（解忧）、生妻财（财源） |
| **妻财** | 我克、获取 | 妻子（男）、财务人员 | 金钱、财物、报酬 | 克父母（损文书）、生官鬼 |
| **官鬼** | 克我、约束 | 丈夫（女）、上司、官方 | 功名、官职、疾病 | 克兄弟、生父母 |
| **父母** | 生我、庇护 | 父母、师长 | 文书、房屋、车船 | 克子孙（损丁）、生兄弟 |

### 2.2 动态吉凶（🔸校准层）

同一六亲，在不同占问中吉凶翻转——这是通用占卜最核心的认知框架之一：

| 六亲 | 吉的场景 | 凶的场景 |
|------|---------|---------|
| 子孙 | 问健康（制病）、求财（财源） | 求官（子孙克官，损功名） |
| 妻财 | 求财（直接用神）、男问婚姻 | 问文书（财克父母） |
| 官鬼 | 求官（直接用神）、女问婚姻 | 问健康（病象） |
| 父母 | 问学业、文书、房产 | 问子女（父母克子孙） |
| 兄弟 | 问友谊、竞技 | 求财（劫财） |

### 2.3 六亲场景语义矩阵（🔸校准层）

| 六亲 | 事业卦 | 感情卦 | 求财卦 | 健康卦 |
|------|--------|--------|--------|--------|
| **兄弟** | 同事、竞争者 | 情敌、第三者 | 花钱、损耗 | 同病、消耗 |
| **子孙** | 下属、创新 | 子女、快乐 | 利润、收益来源 | 药神（克官鬼=克病） |
| **妻财** | 薪资、回报 | 女方（男问） | 钱、投资标的 | 饮食 |
| **官鬼** | 领导、功名、压力 | 男方（女问） | 风险、阻碍 | 病灶 |
| **父母** | 公司、制度、文件 | 长辈意见、家庭压力 | 合同、手续 | 治疗方案、药物 |

## 3. 六神象义矩阵

六神为每一爻附加象征层，根据日干分配。

### 3.1 核心象义

| 六神 | 五行 | 方位 | 正面 | 负面 |
|------|------|------|------|------|
| **青龙** | 木 | 东 | 好消息、贵人、财喜 | （极少负面） |
| **朱雀** | 火 | 南 | 考试顺利、文书有成 | 口舌是非、文书出错 |
| **勾陈** | 土 | 中 | 稳定、不动产 | 拖延、阻碍、停滞 |
| **螣蛇** | 土 | 中 | 灵感、意外转机 | 虚惊、纠缠不清 |
| **白虎** | 金 | 西 | 果断、效率 | 伤灾、血光、暴力 |
| **玄武** | 水 | 北 | 智谋、隐蔽优势 | 欺骗、盗窃、暧昧 |

### 3.2 六神临用神

- **青龙临用神**：所问之事有吉庆之象，过程顺利
- **朱雀临用神**：事涉文书口舌，注意言辞和书面材料
- **勾陈临用神**：事情拖延迟滞，需耐心
- **螣蛇临用神**：事情有不确定性，可能出现意外变化
- **白虎临用神**：事涉刚决果断，需防伤损
- **玄武临用神**：事情有隐蔽面，防欺骗或信息不透明

### 3.3 六神与世应

- 临**世爻**：反映问卦人自身状态
- 临**应爻**：反映对方的状态
- 临**用神**：反映所问之事本身的性质

## 4. 体用生克通用判断

### 4.1 基本原则

- **体卦** = 问卦人自身
- **用卦** = 所问之事 / 外部环境
- 动爻所在的经卦为用卦，另一个为体卦

### 4.2 五行生克

| 关系 | 判断 | 通用释义 |
|------|------|---------|
| **体克用** | 吉 | 我能掌控局面，事情朝有利方向发展 |
| **体生用** | 小凶 | 我在向外付出/消耗，精力或资源外流 |
| **用克体** | 凶 | 外部压力大于承受力，面临阻碍或损失 |
| **用生体** | 吉 | 外部环境有利，有助力或回报流入 |
| **体用比和** | 平 | 势均力敌，发展平稳但无明显突破 |

### 4.3 旺衰修正

在体用关系基础上叠加旺衰信息：

- **体旺 + 体克用** → 极吉
- **体旺 + 用克体** → 虽有困难但有抵抗力
- **体衰 + 体克用** → 可成但过程不顺
- **体衰 + 用克体** → 极凶

### 4.4 体用关系的场景化修正（🔸校准层）

**同一体用关系在不同场景下含义有差异。** 以下为关键校准点：

#### 体克用

| 场景 | 校准后含义 | 注意事项 |
|------|-----------|---------|
| 求财 | 吉——我克财为得财 | 最直接的正面信号 |
| 事业 | 吉——有能力完成，可以争取到 | |
| 谈判 | 吉——我方占优势 | |
| 感情 | **偏中性**——我方主导，但对方可能感到压力 | 桃花占中意味着需要主动出击，但不宜太强势 |
| 健康 | 小吉——身体有抵抗力，但也意味着在消耗对抗 | |

#### 用生体

| 场景 | 校准后含义 |
|------|-----------|
| 求财 | 大吉——财来找我，贵人助力 |
| 感情 | 大吉——对方主动，对我有好感 |
| 事业 | 吉——有助力、有推荐、环境有利 |

#### 体生用

| 场景 | 校准后含义 |
|------|-----------|
| 求财 | 不利——花钱多、投入大于回报 |
| 感情 | 偏不利——我方付出多，容易一厢情愿 |
| 合作 | 偏不利——我方资源流出，对方获益更多 |

#### 用克体

| 场景 | 校准后含义 |
|------|-----------|
| 求财 | 凶——有破财风险或投入大于回报 |
| 感情 | 不利——对方可能给我方带来压力或束缚 |
| 事业 | 不利——上级/环境压力大，需要顶住 |
| 健康 | 凶——注意休息，外部因素影响身心 |

### 4.5 互卦与变卦辅助

- **互卦**：事态发展过程中的内在阻碍或助力
- **变卦**：事态的最终走向和结果趋势

三卦合看：本卦（当前状态）→ 互卦（过程）→ 变卦（结果）。

## 5. 世应关系判断

### 5.1 基本含义

- **世爻** = 问卦人、主方、己方
- **应爻** = 对方、客方、事情的对象

### 5.2 世应生克

| 关系 | 含义 |
|------|------|
| 世生应 | 我主动付出，对方受益 |
| 应生世 | 对方主动帮助或配合 |
| 世克应 | 我占优势，能压制对方 |
| 应克世 | 对方占优势，我被动 |
| 世应比和 | 双方势均力敌 |
| 世应相合 | 双方有缘分/合作意向 |
| 世应相冲 | 双方冲突/不合 |

## 6. 动变分析

### 6.1 动爻的意义

动爻代表卦中**正在发生变化**的力量，揭示推动事态发展的核心因素。

### 6.2 分析要点

1. **谁在动**：动爻是什么六亲 → 变化的驱动力来源
2. **动后变成什么**：变爻的六亲 → 变化的方向
3. **动爻与用神的关系**：生用神则助事，克用神则碍事
4. **动爻与世爻的关系**：生世则有利，克世则不利

### 6.3 各六亲动爻通用含义

- **官鬼动**：有外部压力或权威介入
- **妻财动**：财务层面有变化，或所求之物/事正在移动变化
- **父母动**：文书、契约、庇护力量在变化
- **子孙动**：解决方案在出现，或享乐因素在变化
- **兄弟动**：竞争加剧，或有同辈干扰

### 6.4 动爻爻位通用含义

| 爻位 | 通用含义 |
|------|---------|
| **初爻** | 基础、起始、最底层；事情刚开始萌动 |
| **二爻** | 内卦中位；与家庭、内部事务、基础条件相关 |
| **三爻** | 内外交界；过渡区域，事情在内外之间摇摆 |
| **四爻** | 外卦初位；事情开始进入外部领域 |
| **五爻** | 外卦中位；核心外部因素、决策层、关键人物 |
| **上爻** | 最高位；事态极端化、最终结果、外力介入 |

## 7. 旺衰与力量分析

### 7.1 五行旺衰（月建）

| 状态 | 与月建关系 | 力量 |
|------|-----------|------|
| **旺** | 同五行 | 最强，当令得时 |
| **相** | 被月建生 | 次强，得助力 |
| **休** | 生月建 | 中性偏弱，力量外泄 |
| **囚** | 被月建克 | 弱，受制无力 |
| **死** | 克月建 | 最弱，力不从心 |

### 7.2 日辰的作用

- 月建管**旺衰**（整体力量等级）
- 日辰管**生克冲合**（具体事件触发）
- 日辰冲爻：静爻逢冲「暗动」，动爻逢冲「冲散」
- 日辰合爻：被「绊住」，暂时无法发挥作用

### 7.3 用神旺衰通用判断

| 用神状态 | 通用含义 |
|----------|---------|
| 旺相 | 所问之事有力、可成、趋势正面 |
| 休囚 | 力弱、难成、需等待或借助外力 |
| 被日月冲克 | 短期面临挫折 |
| 得日月生扶 | 短期有助力 |
| 动而化进 | 事态向好 |
| 动而化退 | 事态走弱 |
| 动而化克 | 事情反转，注意变故 |

## 8. 伏神分析

### 8.1 何时需要

用神对应的六亲在卦中六爻里没有出现时，
用神处于「伏藏」状态，需在飞神下寻找。

### 8.2 飞伏关系

| 关系 | 含义 |
|------|------|
| 飞生伏 | 有保护/帮助，所问之事被暗中支持 |
| 飞克伏 | 被压制，短期难以显现 |
| 伏克飞 | 有冲破压制的趋势，终将突破 |
| 伏生飞 | 紧贴其中，难以分离 |
| 飞伏比和 | 暂时搁置，需等待时机 |

### 8.3 飞神六亲对应的遮蔽/阻碍性质

| 飞神六亲 | 阻碍性质 | 实际场景 |
|----------|---------|---------|
| 兄弟 | 同类竞争/干扰 | 竞争者压制、同事争夺 |
| 父母 | 制度/文书阻碍 | 手续未完、审批卡住 |
| 子孙 | 享乐/分心 | 因其他事情分心、自满 |
| 官鬼 | 外部压力/权威阻碍 | 上级阻止、政策限制 |
| 妻财 | 利益纠葛 | 因金钱/利益关系被卡住 |

### 8.4 通用含义

- 所问之事**尚未显现**或**条件尚不成熟**
- 需等待特定的冲、合、生扶条件触发
- 并非一定是凶——只是「时机未到」
- 出伏条件：飞神被冲开之日、伏神得生扶之时

## 9. 方位与环境推断

### 9.1 八卦方位对照

| 卦 | 方位 | 环境特征 |
|----|------|---------|
| 乾 | 西北 | 高处、金属旁、圆形物附近、贵重物品存放区 |
| 兑 | 西 | 近水、有缺口的地方、低洼处、娱乐场所 |
| 离 | 南 | 光亮处、电器旁、窗户附近、文化/教育场所 |
| 震 | 东 | 门口附近、树木旁、经常走动的区域、交通要道 |
| 巽 | 东南 | 走廊/过道、木质家具附近、商业/流通区域 |
| 坎 | 北 | 暗处、水附近、隐蔽处、危险区域 |
| 艮 | 东北 | 角落、门后、储藏室、山/高地 |
| 坤 | 西南 | 地面、柔软处、容器内部、平坦开阔处 |

### 9.2 方位判断优先级

1. **用卦方位**（最高）：所问之事的方向或趋势方向
2. **变卦方位**（次高）：事态发展的方向
3. **体卦方位**（参考）：问卦人自身的位置或出发点
4. **互卦方位**（辅助）：过程中可能经过的方向

### 9.3 方位在不同场景下的使用（🔸校准层）

- **寻物**：用卦方位为主，指示物品所在方向
- **出行**：变卦方位指示利行方向或需注意的方向
- **事业**：方位可对应办公位置、分支机构、合作方向
- **居住**：方位对应房屋/办公室中的区域

### 9.4 六神方位补充

用神所临六神的方位可作为辅助参考，与梅花方位交叉验证：
青龙=东，朱雀=南，勾陈/螣蛇=中，白虎=西，玄武=北

### 9.5 复合环境

上下卦组合描述环境层次：
- **上卦**：外在特征、表面状态
- **下卦**：内在特征、底层状态

## 10. 应期推算通用规则

### 10.1 梅花应期

- **体旺用衰**：应期近，取生体之卦数为时间单位
- **体衰用旺**：应期远，取帮扶体卦的五行当令之时
- **体用比和**：以卦数为应期基本单位
- 卦数对应：乾1 兑2 离3 震4 巽5 坎6 艮7 坤8

### 10.2 六爻应期

- 用神旺相 → 应在逢冲之日/月
- 用神休囚 → 应在生旺之月日
- 用神被冲 → 应在合日（解冲）
- 用神被合 → 应在冲日（解合）
- 用神伏藏 → 应在飞神被冲开之日

### 10.3 表达原则

- 应期判断**不应过于精确**——给出时间范围而非具体某天
- 时间单位需根据事情性质灵活判断：日常小事取时/日，重大决策取月/年
- 同时给出梅花和六爻两套判断，一致则增强可信度

## 11. 交叉验证通用框架

### 11.1 五维度模型

| 维度 | 梅花依据 | 六爻依据 |
|------|---------|---------|
| 总体吉凶 | 体用关系 + 旺衰 | 用神旺衰 + 世应关系 |
| 方位/方向 | 用卦/变卦方位 | 用神临六神方位 |
| 核心动态 | 动爻归属(体/用) | 动爻六亲/六神 |
| 潜在风险 | 互卦/变卦克体 | 伏神/月破/日冲/官鬼动 |
| 时间判断 | 卦数应期 | 地支冲合应期 |

### 11.2 一致性评估

| 标签 | 条件 | 含义 |
|------|------|------|
| **一致** | 两套体系评估方向相同且强度相近 | 高确信度 |
| **互补** | 两套体系从不同角度给出信息，无方向矛盾 | 中确信度，需综合 |
| **不同** | 两套体系评估方向或强度有实质差异 | 低确信度，以六爻为准但呈现两种信号 |

### 11.3 矛盾处理原则

梅花与六爻矛盾时：**以六爻为准**（更精细）。
但在报告中如实呈现两个体系的信号差异。

### 11.4 scene_notes 机制

交叉验证引擎在检测到可能因场景产生歧义的评分点时，
会在输出中附加 `scene_notes` 字段提示 Claude。
Claude 应结合占问内容和本文档的校准层内容进行判断。

## 12. 解读风格与认知标记

### 12.1 三层认知标记（硬性要求）

**绝对不可以把推测当事实呈现。**

| 层级 | 确定性 | 引入语 | 含义 |
|------|--------|--------|------|
| **卦象判断** | 高 | "从卦象来看"、"体用关系显示" | 从卦象数学结构直接得出 |
| **卦象推断** | 中 | "卦象中X的象义指向"、"结合传统象义" | 基于传统象义的延伸 |
| **情境推测** | 低 | "结合你的描述推测"、"一种可能是" | 结合用户情况的主观推测 |

### 12.2 叙事原则

- **具象化**：将抽象卦象符号翻译为用户能理解的具体场景
- **因果链条**：每个判断展示"卦象信号 → 传统释义 → 当前情境下的含义"
- **坦承不确定性**：信号模糊或矛盾时如实说明
- **实用导向**：解读最终落实为可执行的行动或关注方向

### 12.3 禁止事项

- 把情境推测包装成卦象判断
- 只说"吉"/"凶"不解释依据
- 给出过于精确的预测（如"3月21日下午会成"）
- 用万金油式表述应付（如"有好有坏，注意把握"）
- 在没有卦象依据的情况下给出方向性建议

## 13. 特殊情形处理

### 13.1 占问内容模糊

用户说"帮我算一卦"但不说具体问什么：必须追问。
没有明确的占问内容就无法确定用神。

### 13.2 一事不二占

短时间内对同一问题多次起卦，传统上被认为不当。
告知原则，建议换角度追问，但若用户坚持则尊重意愿。

### 13.3 涉及他人隐私

保持中立客观，只基于卦象分析关系动态，不做人格评判。

### 13.4 涉及健康

只提供方向性参考，**必须提醒**占卜不能替代医学诊断。

### 13.5 涉及重大决策

提供卦象分析作为参考维度之一，**必须提醒**重大决策应综合多方面因素。

### 13.6 八纯卦

上下卦相同时：方位信息单一但高度集中，环境特征加倍强调。

### 13.7 游魂卦 / 归魂卦

- **游魂卦**：事态飘忽不定，可能有远行或变动之象
- **归魂卦**：事态回归稳定，有定论或回到原点之象

### 13.8 六冲卦 / 六合卦

- **六冲卦**：事态动荡，难以稳定；但病卦遇六冲可能好转（冲散病气）
- **六合卦**：事态和合，有利于达成目标；但若问急事则可能拖延（"合住"不动）

## 15. 用神场景分析指南（🔸校准层）

> 本章为六类核心占问场景提供具体的分析指引。
> Claude 根据占问内容匹配相应场景，但不限于以下分类——
> 这些是「示例」而非「穷举」。

### 15.1 求财/投资

**用神：** 妻财爻

**分析重点：**
- 妻财旺衰 → 财气强弱
- 子孙爻（财源）→ 子孙旺且动 = 持续生财
- 兄弟爻（劫财）→ 兄弟旺且动 = 有人分利或耗财
- 官鬼爻（风险）→ 官鬼旺且动 = 投资有风险/阻碍
- 世爻旺衰 → 我方能否把握住财

**校准要点：**
- 财旺生世 → 可得财，且来得容易
- 财旺克世 → 财多但求之不得，或得到后带来负担
- 财衰 → 财气不足，此时不宜冒进
- 兄弟持世 → 主动劫财之象，破财风险大

### 15.2 事业/功名

**用神：** 官鬼爻

**分析重点：**
- 官鬼旺衰 → 功名是否可得
- 父母爻（贵人/文书）→ 旺则有提携，有正式文书确认
- 子孙爻动否 → 子孙克官，不利功名（自由散漫之象）
- 世爻旺衰 → 我方能力和状态

**校准要点：**
- 官旺生世 → 有功名可得，上级认可
- 官旺克世 → 压力大，但「有官星才有官做」，不全是坏事
- 官衰 → 事业动力不足，时机未到

### 15.3 感情/婚姻

**用神：** 男问取妻财爻（代表女方），女问取官鬼爻（代表男方）

**分析重点：**
- 用神旺衰 → 对方的状态和态度
- 世应关系 → 两人之间的互动模式
- 六神色彩 → 情感层面的细节
- 兄弟爻（情敌/第三者）→ 兄弟旺动要警惕

**校准要点（🔸重点校准区域）：**
- 用神旺且生世 → 对方主动，感情最理想的信号
- 用神旺且克世 → 对方强势，关系中我方弱势
- 世应相合 → 有缘分基础，但不等于一定在一起
- 世应相冲 → 目前有矛盾或不合
- 世生应 → 我追对方；应生世 → 对方追我
- 兄弟动 → 有竞争者介入
- 动爻化妻财/化官鬼 → 事态正在向感情方向发展
- **体克用在感情中 → 偏中性**：我方掌控力强，但需注意不要让对方感到压力。
  这是最容易被误判为"大吉"的场景——实际上"掌控"在感情中不等于"顺利"

### 15.4 健康

**用神：** 世爻（自问）或对应六亲（问他人）

**分析重点：**
- 官鬼为「病」，子孙为「药」
- 官鬼旺衰 → 病势轻重
- 子孙旺衰 → 治疗是否有效

**校准要点：**
- 官鬼旺且克世 → 病势较重
- 子孙旺且动 → 有药可解，治疗有效
- 官鬼化进神 → 病情在发展
- 官鬼化退神 → 病情在缓解

### 15.5 出行/迁移

**分析重点：**
- 世爻为出行人，应爻为目的地
- 世旺 → 出行顺利
- 世衰被克 → 不宜出行或有阻碍
- 动爻方向 → 提示路线上的变数
- 变卦趋势 → 出行后的结果

### 15.6 决策（A 或 B）

当用户面临二选一的决策时：

- 不直接说"选A"或"选B"
- 分析当前态势下各个方向的能量分布
- 体旺的方向行动力强，体衰的方向需要更多准备
- 变卦趋势指示事态的自然走向
- **选择权在用户手中——卦象提供信息，不是命令**

## 16. 解读伦理与边界

### 16.1 不做确定性承诺

卦象分析提供的是**概率性的方向指引**，不是确定性的预言。
所有判断都应保持适度的开放性。
"卦象指向X方向"不等于"X一定会发生"。

### 16.2 不替用户做决定

卦象分析帮助用户**看清当前态势下各个方向的能量分布**。
最终决策权在用户手中。尤其是重大人生决策（婚姻、辞职、大额投资等），
应明确提醒"卦象仅供参考，请结合自身实际情况综合判断"。

### 16.3 不介入的领域

- 涉及生死判断 → 只分析健康趋势，不做寿命断言
- 涉及第三方隐私 → 只分析卦象中的信号，不做人肉搜索
- 涉及违法行为 → 不提供任何帮助

### 16.4 积极建设性原则

即使卦象显示不利，也应当给出**建设性的应对建议**。
"不宜此时行动"之后，应跟上"那么何时/何种条件下更合适"。
卦象分析的目的是帮助用户做出更好的决策，而非制造焦虑。
