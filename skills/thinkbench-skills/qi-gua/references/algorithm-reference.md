## 三、时间起卦算法

### 前置步骤：公历→农历转换

这是**最关键**也是**最容易出错**的步骤。梅花易数时间起卦使用的月、日数据
必须是**农历月和农历日**，而非公历月日。

由于农历与公历的对应关系无法用简单公式推算（涉及闰月、大小月等不规则因素），
在无 Python 环境时：

1. **优先使用网络搜索**确认具体日期的农历对应
2. 若无法搜索，使用以下粗略估算（注意误差可能达1个月或数天）：
   - 农历月≈公历月减1（仅为极粗略估算，闰月年误差更大）
   - 这种估算**不可靠**，务必告知用户
3. **明确标注**使用了哪种方式获取农历日期

### 计算公式

```
上卦数 = (年支数 + 农历月 + 农历日) mod 8    [余0作8]
下卦数 = (年支数 + 农历月 + 农历日 + 时辰数) mod 8    [余0作8]
动爻位 = (年支数 + 农历月 + 农历日 + 时辰数) mod 6    [余0作6]
```

根据先天八卦数对应表（第一节），将上卦数和下卦数转换为卦名。

### 计算示例

**起卦时间：** 2026年3月15日 14:52（公历）

1. **农历转换**（需确认）：假设为农历丙午年正月廿六
2. **年支**：2026年为丙午年，年支=午，午数=7
3. **农历月**：正月=1
4. **农历日**：廿六=26
5. **时辰**：14:52为未时，未数=8

```
上卦 = (7 + 1 + 26) mod 8 = 34 mod 8 = 2 → 兑
下卦 = (7 + 1 + 26 + 8) mod 8 = 42 mod 8 = 2 → 兑
动爻 = (7 + 1 + 26 + 8) mod 6 = 42 mod 6 = 0 → 作6 → 第6爻
```

本卦：兑为泽（兑上兑下），动爻第6爻

## 四、报数起卦算法

> 注意：此方法仅在用户主动要求时使用。

### 单数起卦

```
上卦 = 数字 mod 8    [余0作8]
下卦 = 当前时辰数 mod 8    [余0作8]
动爻 = 数字 mod 6    [余0作6]
```

### 双数起卦

```
上卦 = 第一个数 mod 8    [余0作8]
下卦 = 第二个数 mod 8    [余0作8]
动爻 = (第一个数 + 第二个数) mod 6    [余0作6]
```

---

## 五、汉字起卦算法

> 注意：此方法仅在用户主动要求时使用。

### 单字

```
上卦 = 笔画数 mod 8    [余0作8]
下卦 = 笔画数 mod 8    [余0作8]（同上卦）
动爻 = (笔画数 + 时辰数) mod 6    [余0作6]
```

### 两字

```
上卦 = 第一字笔画 mod 8
下卦 = 第二字笔画 mod 8
动爻 = (总笔画 + 时辰数) mod 6
```

### 多字（三字及以上）

```
前半笔画总和取上卦，后半笔画总和取下卦
动爻 = (全部笔画总和 + 时辰数) mod 6
奇数字时，前半取 n//2 个字，后半取剩余
```

## 六、变卦推算

动爻所在位置的爻发生阴阳变化：
- 阳爻（———）→ 变为阴爻（— —）
- 阴爻（— —）→ 变为阳爻（———）

其余爻不变。变化后的六爻重新组成一个新的卦，即为**变卦**。

**步骤：**
1. 写出本卦的六爻（从下到上，第1爻到第6爻）
2. 将动爻位对应的爻做阴阳互换
3. 下三爻（1-3）查八卦表得变卦下卦
4. 上三爻（4-6）查八卦表得变卦上卦

## 七、互卦推算

互卦取本卦的中间四爻重新组合：

```
互卦下卦 = 本卦第2、3、4爻（从下到上）
互卦上卦 = 本卦第3、4、5爻（从下到上）
```

注意：第3、4爻在上下互卦中被重复使用，这是正确的。

## 八、体用判定

**核心规则：** 动爻所在的经卦（上卦或下卦）为**用卦**，另一个经卦为**体卦**。

```
动爻在第1、2、3爻 → 下卦为用，上卦为体
动爻在第4、5、6爻 → 上卦为用，下卦为体
```

**体卦**代表问卦人（自身），**用卦**代表所问之事（对象、环境）。

## 十一、五行生克关系

### 五行相生

金 → 水 → 木 → 火 → 土 → 金（循环）

### 五行相克

金 → 木 → 土 → 水 → 火 → 金（循环）

### 体用关系判断

| 关系 | 含义 | 通用判断 |
|------|------|---------|
| 体克用 | 体强而制用 | 吉，主动权在我方 |
| 体生用 | 体泄气于用 | 费力，需付出 |
| 用克体 | 用强而制体 | 不利，受制于外 |
| 用生体 | 用助力于体 | 吉，顺势可成 |
| 比和 | 同五行 | 中平，可成但需时间 |

四季末月（辰、未、戌、丑月，即农历3、6、9、12月）土旺。

### 旺衰的通用影响

- **体旺**：问卦人状态好，有利于推进所求之事
- **体衰**：问卦人状态弱，不利，需借助外力或等待时机
- **用旺**：所问之事/对方力量强，环境状态活跃
- **用衰**：所问之事/对方力量弱，推进阻力较小但资源不足
