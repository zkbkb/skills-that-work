# 五行物品归属推理方法论

> 本文档提供判断任意物品五行属性的系统化推理框架。
> 当占问内容涉及具体物品（寻物、购买、投资标的、礼物选择等）时，
> 可用本方法论将物品的五行属性与卦象进行交叉验证。

---

## 一、三级判断法则

判断任何物品的五行属性，按以下优先级逐级判断：

### 第一级：材质优先

物品的物理材质是最直观、最可靠的五行归属依据。

| 材质 | 五行 | 对应卦 | 常见物品举例 |
|------|------|--------|------------|
| 金属（铁、钢、铜、铝、金、银） | 金 | 乾、兑 | 钥匙、硬币、首饰、刀具、金属工具 |
| 木质、纸质、竹质、皮革 | 木 | 震、巽 | 书籍、证件、木梳、皮带、纸币 |
| 塑料（石油衍生物）、电子元件、发光物 | 火 | 离 | 手机、电脑、灯具、充电器 |
| 陶瓷、砖石、泥土、混凝土 | 土 | 坤、艮 | 花盆、瓷器、砖块、水泥构件 |
| 液体、玻璃（透明如水）、冰 | 水 | 坎 | 水杯（盛水时）、酒瓶、墨水 |
| 织物、棉麻、布帛 | 土 | 坤 | 衣服、毛巾、床单、窗帘 |

**复合材质处理：**
- 以**主体/外壳材质**为主判断
- 若主体和核心功能材质不同，两者都记录，以材质为主、功能为辅
- 例：带皮套的手机 → 手机本体为火（电子），皮套为木（皮革），综合取**火**

### 第二级：功能其次

当材质不明确或物品以功能定义时，以其核心功能的五行属性判断：

| 功能类别 | 五行 | 推理逻辑 | 举例 |
|----------|------|---------|------|
| 切割、锋利、决断 | 金 | 金主肃杀、决断 | 剪刀、刮胡刀 |
| 生长、延展、连接 | 木 | 木主生发、伸展 | 绳索、数据线 |
| 通讯、照明、展示 | 火 | 火主光明、传播 | 电话、投影仪、显示屏 |
| 承载、存储、包容 | 土 | 土主承载、容纳 | 箱子、背包、收纳盒 |
| 流通、清洁、运输 | 水 | 水主流动、润下 | 水管、洗衣机 |
| 财务、价值交换 | 金 | 金主价值、财富 | 信用卡、银行卡 |
| 记录、文化、教育 | 木 | 木主文化、生长 | 笔记本、教材 |
| 保暖、防护 | 土 | 土主厚德、包容 | 外套、手套、头盔 |

### 第三级：象征辅助

当前两级均无法清晰判断时，借助易学传统中的象征体系：

| 象征维度 | 五行归属逻辑 |
|----------|------------|
| 形状为圆 | 乾金（乾为圆） |
| 形状为方 | 坤土（坤为方） |
| 形状为长条 | 巽木（巽为绳直） |
| 颜色偏白/银 | 金 |
| 颜色偏红/紫 | 火 |
| 颜色偏青/绿 | 木 |
| 颜色偏黑/深 | 水 |
| 颜色偏黄/棕 | 土 |
| 与老人/权威相关 | 乾金 |
| 与年轻女性相关 | 兑金 |
| 与母亲/大地相关 | 坤土 |

## 二、常见争议物品的辨析

有些物品的五行归属在实践中存在分歧，以下为推荐判断及其理由：

### 手机 → 火

- **材质层面**：含金属（金）+ 玻璃（水）+ 塑料/电子（火）
- **功能层面**：通讯、发光、显示 → 火
- **结论**：以核心功能"电子通讯与显示"取火，对应离卦
- **注意**：若强调手机的金属外壳（如全金属机身），可兼取金

### 钱包 → 木/土（需区分）

- **皮质钱包**：皮革为动物制品，传统归木 → 取木，巽卦
- **布质钱包**：布帛为坤土之象 → 取土，坤卦
- **功能视角**：容器/收纳功能 → 土
- **结论**：皮质取木，布质取土，以材质为主

### 钥匙 → 金

- 金属材质，无争议 → 金，兑/乾
- 若为塑料钥匙扣附带的钥匙，仍以钥匙本体（金属）为主

### 眼镜 → 火/金

- **金属框**：金
- **塑料框**：火
- **功能（视觉/光学）**：离火
- **结论**：金属框取金，塑料框取火；功能上统一借用离卦

### 雨伞 → 水/木

- **与雨水相关**：水
- **伞骨/伞柄为木或金属**：木或金
- **功能（遮蔽、抵御）**：坎水（坎为险、为防护）
- **结论**：以功能场景取水为主，兼顾材质

### 药品 → 土/木

- **中药、草药**：木
- **西药（化学制品）**：土（土主化生万物）
- **功能（治疗、修复）**：土（土主承载、滋养）
- **结论**：草药取木，化学药品取土

## 三、行业/领域五行速查

当占问涉及行业选择、事业方向时，可参考以下归属：

| 行业/领域 | 五行 | 推理逻辑 |
|-----------|------|---------|
| 金融、银行、保险、证券 | 金 | 价值交换、金属货币 |
| 军事、警察、法律、执法 | 金 | 肃杀、决断 |
| 教育、出版、文化、研究 | 木 | 生长、文化 |
| 医疗、养生、中医 | 木 | 生命、仁慈 |
| IT、互联网、传媒、广告 | 火 | 传播、照明 |
| 餐饮、娱乐、影视 | 火 | 光明、热闹 |
| 房地产、建筑、农业 | 土 | 承载、大地 |
| 物流、仓储、人力资源 | 土 | 包容、运转 |
| 贸易、运输、旅游、航运 | 水 | 流动、周转 |
| 咨询、策划、智库 | 水 | 智慧、谋略 |

## 四、五行归属与卦象交叉验证

确定物品的五行属性后，应与起卦结果进行交叉验证：

### 4.1 验证逻辑

- 物品属金 → 卦中应见兑或乾（若不见，可能暗示物品状态或环境已偏离常态）
- 物品属火 → 卦中应见离
- 物品属木 → 卦中应见震或巽
- 物品属水 → 卦中应见坎
- 物品属土 → 卦中应见坤或艮

### 4.2 不一致时的处理

若物品五行与卦象中的相关卦位不匹配：

1. **重新审视物品的五行归属**——是否存在更合理的归属方式？
2. **检查是否有"暗含"**——例如互卦中是否包含对应的经卦？
3. **将不一致作为信息**——不一致本身可能暗示物品已不在原始状态、
   环境发生了变化、或有其他因素介入

### 4.3 报告中的表述

在分析报告中应说明：
- 物品的五行归属判断（及其推理过程）
- 与卦象的对应/不对应关系
- 不对应时的可能解读
